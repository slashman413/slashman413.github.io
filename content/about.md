<p><strong>intelligent agents</strong> (<a href="https://slashmantools.us/">slashmantools.us</a>) is an independent,
one-person project run under the handle <strong>slashman413</strong> — also known as
<strong>「intelligent agents」</strong>. It is built and maintained by a single developer who works in software and
enjoys automating tedious tasks. The site combines two things I use every day: a growing collection of
<strong>free, fast, privacy-first web tools</strong>, and a small library of <strong>practical, hands-on
guides</strong> on AI, development, and personal finance.</p>

<h2>Our mission</h2>
<p>Most online utilities are cluttered with pop-ups, demand file uploads to a server, or hide basic features
behind a sign-up wall. The goal here is the opposite: clean, single-purpose tools that load instantly, work
on mobile, support multiple languages, and respect your privacy by doing the work locally in your browser
whenever it is technically possible. No account is ever required to use a tool.</p>

<h2>What we cover</h2>
<ul>
  <li><strong>AI helpers</strong> — LLM VRAM calculator, token &amp; API-cost calculator, AI image-size
      calculator, and a prompt library.</li>
  <li><strong>Developer tools</strong> — JSON formatter &amp; regex tester, Base64/JWT/UUID/hash utilities,
      and color tools.</li>
  <li><strong>Everyday utilities</strong> — password generator, word counter, unit converter, QR code
      generator, image compressor, PDF tools, calculators, and a Pomodoro timer.</li>
  <li><strong>Finance &amp; data dashboards</strong> — Taiwan ETF analysis, a stock screener with
      backtesting, a compound-interest calculator, and a 3D global-events tracker.</li>
  <li><strong>Guides &amp; articles</strong> — short, practical explainers (for example, how to estimate LLM
      token cost) written to answer a specific question and then get out of your way.</li>
</ul>

<h2>How the tools work</h2>
<p>The tools are hand-built static web apps. Where a tool processes your input — text, files, or images —
that processing happens <strong>client-side in your browser</strong>, so the data you enter never leaves
your device. The finance dashboards are generated from publicly available market data and refreshed
automatically. Because there is no server-side storage of your inputs, there is nothing for us to see, sell,
or lose.</p>

<h2>Editorial approach</h2>
<p>I write the guides from first-hand experience — I build the tool, use it on my own projects, and only
then document what actually worked. Where numbers or prices change over time (such as API rates), the goal
is to explain the method so you can re-check current figures yourself, rather than trusting a stale table.
Nothing on this site is professional financial, medical, or legal advice; the finance dashboards are
research aids, not recommendations (see the <a href="terms.html">Terms of Service</a>). When I get something
wrong, I'd genuinely like to hear about it so it can be fixed.</p>
